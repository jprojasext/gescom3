En el marco de la Junta Arbitral Nacional de Consumo del Gobierno de España, se garantiza la protección de los datos personales proporcionados. La presente declaración de protección de datos tiene como objetivo informar sobre cómo se recopila, utiliza, divulga y protege la información proporcionada a través de este sitio web.


Durante el proceso de solicitud de arbitraje, se pueden solicitar ciertos datos personales, como nombre, dirección de correo electrónico, número de teléfono, entre otros. Estos datos se recopilan de manera voluntaria y con el consentimiento expreso del interesado/a.


La información recopilada se utiliza únicamente con el propósito de gestionar las solicitudes de arbitraje y brindar los servicios correspondientes. Los datos personales proporcionados no se utilizarán para ningún otro fin sin el consentimiento previo del interesado/a.


Se implementarán medidas adecuadas para garantizar la confidencialidad de los datos personales. No se divulgarán, venderán ni transferirán los datos personales a terceros sin el consentimiento del interesado/a, a menos que sea requerido por ley o por una autoridad competente.


Se implementarán medidas de seguridad para proteger los datos personales contra el acceso no autorizado, la divulgación o la destrucción no autorizada. Se revisan y actualizan regularmente las prácticas de seguridad para mantener la confidencialidad e integridad de los datos personales.


Como titular de los datos personales, se tienen ciertos derechos según la legislación vigente. Estos derechos incluyen el acceso, rectificación, cancelación y oposición al tratamiento de los datos personales.
